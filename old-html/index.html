FAQ

Questions?
We've Got Answers.

On Chrome, our extension injects text directly into the focused text field using the browser's DOM API. On desktop (Electron app), we use OS-level accessibility APIs — AXUIElement on macOS and UI Automation on Windows. On Android, we use the Accessibility Service + InputConnection API, similar to how Gboard and SwiftKey work.

No. Audio is sent to Groq's API for real-time processing and is immediately discarded after the transcript is generated. We never store audio recordings. On Privacy Mode (or Offline Mode), processing happens 100% on your device — nothing leaves your computer.

Many Indians mix their mother tongue with English mid-sentence. Tanglish is Tamil+English, Tenglish is Telugu+English, Hinglish is Hindi+English. VoiceFlow's AI is specifically tuned to handle this code-switching naturally, preserving both languages exactly as spoken.

Yes! Our Pro plan includes VoiceFlow Edge — an on-device AI pipeline using whisper.cpp (Whisper Small, 75MB) and Llama 3.2 1B (800MB). It works without internet, perfect for rural India or privacy-sensitive users. It auto-activates when network is unavailable.

Accessibility Permission is required so VoiceFlow can detect which text field you're in and type text into it — across ANY app on your device. This is the same permission Gboard, SwiftKey, and every keyboard app requires. We never read your screen content. Our content script is open source for verification.

Yes. Groq offers generous free tiers: 7,200 seconds of Whisper audio/day and 14,400 Llama 3.3 requests/day. A user dictating ~50 times/day at ~20 seconds each uses only ~1,000 seconds — well within free limits. Pro users don't need their own API key; VoiceFlow manages it.
